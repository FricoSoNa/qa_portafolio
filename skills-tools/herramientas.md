# Herramientas y Tecnologías que Uso

Como QA Tester Junior, estas son las herramientas principales que manejo diariamente en mis proyectos y pruebas:

## Pruebas Manuales y Gestión de Bugs
- **Jira** — Reporte, seguimiento y priorización de defectos (severidad, pasos para reproducir, evidencias adjuntas).
- **Google Sheets / Excel** — Creación de checklists, matrices de casos de prueba, tablas de decisión y valores límite.

## Testing de API
- **Postman** — Diseño y ejecución de colecciones REST, validación de respuestas JSON, códigos de estado, esquemas, variables de entorno y scripts de test automáticos básicos.

## Navegadores y Testing Cross-Browser
- **Google Chrome** — Pruebas principales (DevTools para inspección de elementos y red).
- **Mozilla Firefox** — Validación en diferentes motores de renderizado.
- **Microsoft Edge** — Compatibilidad adicional en entornos Windows.

Estoy constantemente practicando y agregando nuevas herramientas según los proyectos del bootcamp.  
Si tienes alguna recomendación o quieres que probemos algo juntos, ¡avísame! 🚀
