# Proyecto: Urban Routes (Web App)

**Contexto**  
Aplicación de transporte compartido. Pruebas enfocadas en flujos clave del bootcamp TripleTen.

**Objetivos de testing**  
- Validar requisitos funcionales  
- Identificar inconsistencias y áreas grises  
- Ejecutar pruebas de humo, regresión y exploratorias

**Flujos probados**  
- Registro y autenticación  
- Agregar licencia de conducir  
- Cálculo de precios y duración en viajes compartidos

**Técnicas usadas**  
- Clases de equivalencia y valores límite  
- Tablas de decisión para combinaciones

**Resultados destacados**  
- Comprobé los diseños de la página comparandolos con los requisitos brindados
- Hice casos de prueba así como listas de comprobación para las funciones de la página
- Realicé los informes de errores respectivos en jira, haciendo un total de 11
