**Contexto**  
Aplicación de transporte compartido. Pruebas enfocadas en flujos clave del bootcamp TripleTen.

**Objetivos de testing**  
- Validar requisitos funcionales  
- Identificar inconsistencias y áreas grises

- Cálculo de precios y duración en viajes compartidos

**Técnicas usadas**  
- Clases de equivalencia y valores límite  
- Tablas de decisión para combinaciones

**Resultados destacados**
